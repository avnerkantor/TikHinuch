Tikhinuch

View project on GitHub

Tik Hinuch (Education Files in Hebrew) is a visual data explorer of educational datasets. It's goal is to help educators, parents, policy makers and researchers to learn from the most important educational resources. The project focuses currently on the OECD PISA datasets.

We used R language to analyze the datsets together with shiny, dplyr and ggplot2 packages. We built a Shiny server in Google Compute Cloud. For the CMS we used Django and it's hosted in Heroku and combines the Shiny application as iframe.
